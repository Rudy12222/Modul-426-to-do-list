# Modul-426-to-do-list
To-Do-Listen-App: Eine kleine Web-App, mit der Nutzer Aufgaben anlegen, beabeiten und löschen können.

Kurzbeschreibung

Die App dient dazu, Aufgaben übersichtlich zu verwalten. Nutzer können neue Tasks erfassen, erledigte Tasks markieren und nicht mehr benötigte Tasks entfernen. Die Aufgaben werden in einer Liste dargestellt.

Warum eignet sich das Projekt für Scrum?

Einfach verständlich (klarer Nutzen und klare Funktionen)

Gut in kleine Schritte teilbar (User Stories, Story Points, Sprints)

Schrittweise erweiterbar (neue Features können nach und nach ergänzt werden)

Kernfunktionen

Aufgabe hinzufügen

Aufgabenliste anzeigen

Aufgabe Bearbeiten

Aufgabe löschen

Als Nutzer möchte ich eine neue Aufgabe hinzufügen, damit ich nichts vergesse.

Als Nutzer möchte ich eine Aufgabe bearbeiten.

Als Nutzer möchte ich meine Aufgaben in einer Liste sehen, damit ich den Überblick behalte.
